## 권한 체계

권한은 아래 순서로 상위 권한이 하위 권한을 모두 포함합니다.

```
root > staff (회장/감독) > manager > member > basic
```

| 권한 | 설명 |
|---|---|
| `root` | 최고 관리자. 모든 기능 사용 가능 |
| `staff` | 회장 또는 감독. 스태프 이하 관리 가능 |
| `manager` | 매니저. 멤버 이하 관리 가능 |
| `member` | 승인된 동아리 멤버 |
| `basic` | 회원가입 후 승인 전 기본 상태 |

## 향후 계획

### 커뮤니케이션 연동
- **카카오톡 채널 연계** — 카카오 비즈니스 채널 연결, 공지사항·일정 알림 자동 발송, 카카오 소셜 로그인
- **이메일 연동** — 멤버 신청 승인/거부 알림 메일, 공지사항 구독 메일 발송 (nodemailer)
- **전화번호 인증** — 회원가입 시 SMS 인증 코드 발송, 마케팅 수신 동의 연동

### 게시판
- **파일 첨부** — 게시글·공지사항 이미지/파일 업로드
- **게시글 수정 로그** — 수정 이력 열람 기능

### 기록·통계
- **경기 기록** — 경기 결과, 타격/투구 기록 입력 및 조회
- **선수 통계** — 시즌별 타율, 방어율 등 개인 기록
- **앨범/갤러리** — 경기·행사 사진 갤러리

### 운영
- **출석 관리** — 훈련 출석 체크 및 현황 조회
- **모바일 최적화** — 반응형 디자인

## 활동 로그 (`activity_logs`)

모든 쓰기/삭제/관리 액션은 `activity_logs` 테이블에 기록됩니다.

| 권한 | 기록되는 액션 |
|---|---|
| member+ | 게시글 작성/수정/삭제, 댓글 작성/수정/삭제 |
| manager+ | 일정 추가/수정/삭제, 멤버 승인/거부, 로스터 추가/수정/삭제 |
| staff+ | 매니저 임명/해제, 멤버 강등, 유저 차단/해제, 연도 설정 |
| root | 스태프 임명/해제 |

로그 열람 권한: 자신보다 권한이 높은 유저만 해당 유저의 로그 탭 열람 가능

## 개인정보 취급

본 서비스는 동아리 운영을 목적으로 아래 개인정보를 수집·이용합니다.

### 수집 항목

| 항목 | 수집 목적 | 보유 기간 |
|---|---|---|
| 아이디, 비밀번호(암호화) | 회원 식별 및 로그인 | 회원 탈퇴 시까지 |
| 이메일 | 계정 관리, 알림 발송 | 회원 탈퇴 시까지 |
| 실명, 학번 | 동아리 멤버 인증 | 회원 탈퇴 시까지 |
| 전화번호 | 본인 인증, 연락 수단 | 회원 탈퇴 시까지 |
| OB/YB 구분 | 동아리 활동 내역 관리 | 회원 탈퇴 시까지 |
| 활동 로그 | 서비스 운영 및 부정 사용 방지 | 회원 탈퇴 시까지 |

### 마케팅 수신 동의

이메일, SMS, 카카오톡을 통한 마케팅 정보 수신은 **선택 동의** 사항입니다.  
회원가입 시 또는 마이페이지에서 항목별로 동의·철회할 수 있습니다.

### 제3자 제공

수집된 개인정보는 동아리 운영 목적 외 제3자에게 제공하지 않습니다.  
단, 법령에 따른 요구가 있는 경우는 예외로 합니다.

### 권리 행사

회원은 언제든지 마이페이지에서 본인의 개인정보를 열람·수정할 수 있으며,  
회원 탈퇴 시 관련 정보는 즉시 삭제됩니다.
