```bash

npm install
# .env 파일 설정 후

# → http://localhost:3001
```


## 현재 구현된 기능

### 공개 페이지
- **홈** — 공지사항 미리보기, 이달의 팀 일정 미니 캘린더
- **선수단 명단** — 선수/스태프/영구결번 필터, 이름·번호 검색
- **팀 일정** — 월별 캘린더 (공휴일 + 팀 일정 표시, 연/월 이동)
- **공지사항** — 목록(고정글 상단), 상세보기, 글쓰기
- **게시판** — 목록, 상세보기, 글쓰기

### 인증 및 권한
- 회원가입 신청 → staff 승인 후 활성화
- JWT 기반 로그인
- 권한 체계: `root` → `staff`(회장/감독) → `manager` → `user`(ob/yb)

### 백엔드 API
- 선수단, 게시판, 공지사항, 팀 일정, 공휴일 CRUD
- 회원가입/로그인/내 정보 조회
- 관리자: 회원 승인·거부, 권한 변경
